# TaskFlow

## Descrição

TaskFlow é uma aplicação web de gerenciamento de tarefas desenvolvida com auxílio da ferramenta de Inteligência Artificial Bolt.new.

A aplicação permite:

- Cadastro de tarefas
- Definição de categoria
- Definição de prioridade
- Controle de prazo
- Marcação de tarefas concluídas
- Exclusão de tarefas
- Dashboard com estatísticas em tempo real
- Gráfico de prioridades
- Busca e filtros

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Bolt.new (IA Generativa)

## Prompt utilizado:

Crie uma aplicação web de produtividade chamada TaskFlow com tema escuro (fundo preto, detalhes dourados), separando o código em arquivos: index.html, style.css e script.js.
A aplicação deve ter duas telas navegáveis por abas no topo:
Aba 1 — Dashboard:
4 cards de estatísticas: progresso geral (%), tarefas pendentes, concluídas e atrasadas
Gráfico de barras mostrando quantidade de tarefas por prioridade (Alta, Média, Baixa)
Lista com as 5 tarefas mais recentes, mostrando nome, status e prioridade
Todos os números atualizam automaticamente conforme o usuário interage
Aba 2 — Tarefas:
Formulário para adicionar nova tarefa com campos: nome, categoria (Pessoal, Estudo, Trabalho, Saúde, Finanças), prioridade (Alta, Média, Baixa) e prazo (data)
Validação: não permitir salvar sem nome
Campo de busca por texto e filtros por prioridade e por status (pendente/concluída)
Lista de tarefas com checkbox para marcar como concluída, badge colorido de prioridade e botão de excluir
Tarefas com prazo vencido devem ter um badge "atrasada" em vermelho
Inclua algumas tarefas de exemplo já cadastradas para demonstração. O código deve ser simples, bem comentado e fácil de entender.

## Como Executar

1. Faça o download do projeto.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
